## github配置流程
### 1. 配置本地密钥
```batch
ssh-keygen -t rsa -C "your_email@youremail.com" 
```
### 2. 配置用户名和地址
```batch
$ git config --global user.name "your name"
$ git config --global user.email "your_email@youremail.com"
```
### 3. 初始化仓库
```batch
git init
```
### 4. 推送
```batch
git add <文件>
git commit -m "提交信息"
git remote git remote add origin <仓库地址>
git push origin master
```
